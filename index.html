GIT COMMANDS:

CONFIGURATION:

git --version
git config --global user.name “name”
git config --global user.email “---”
git config --list

CLONE & STATUS:

git clone -link-
git status

ADD & COMMIT:

git add file_name
git commit -m “message”
git push origin -branch-

INIT COMMANDS:

git remote add origin -link-
git init : create a new repo
git remote -v: to verify remote
git branch : to check branch
git branch -M main: to rename branch
git push origin main
git push -u: set origin main as default for push

BRANCH COMMANDS:

git branch : to check branch
git branch -M main: to rename branch
git checkout : exit branch
git checkout -b -new branch name -
git branch -d -name-: to delete branch

MERGING A BRANCH:

Pull request on GitHub
To reflect local changes on remote repo: git pull origin -branch-

TERMINAL COMMANDS

cd: change directory
cd .. : exit directory
mkdir: create new directory
clear
ls: list files
ls -h: hidden files

WORKFLOW:

Git repo => clone => changes => add => commit => push
